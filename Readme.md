A real-time online Monopoly game with a fully 3D board, multiplayer synchronization, and integrated voice chat.  
Built using React Three Fiber on the frontend and a custom Node.js + Socket.IO backend.


## Tech Stack

### Frontend
- **React** (SPA)
- **React Three Fiber** for 3D rendering
- **@react-three/drei** for model/texture utilities
- **Zustand** for global state management
- **WebRTC** for peer-to-peer voice chat
- **Optimized 3D loading**
  

### Backend
- **Node.js + Express**
- **Socket.IO** for state sync, real-time events, and room management
- **Socket.IO WebRTC signaling**
- **MongoDB** for persistent game states



## Features
- Real-time multiplayer with turn-based logic
- Fully interactive 3D Monopoly board
- Preloaded models/textures for reduced loading time
- Token movement animations
- Built-in voice chat per room
- Custom game engine built with Socket.IO
